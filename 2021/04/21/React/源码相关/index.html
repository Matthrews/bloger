React16.8.0

React

  1. hooks

  2. useReducer 懒初始化

React DOM

  1. 使用 Object.is 比较 useState 和 useReducer 的值

  2. React.lazy() 支持异步

  3. Effect 的清楚函数要么返回 undefined 要么返回 function, 否则是不允许的

React Test

  1. 支持 hooks

  2. 新增 ReactTestRenderer.act() 和 ReactTestUtils.act() 用于批量操作以贴近实际场景
