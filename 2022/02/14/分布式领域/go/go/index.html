Go 语言学习笔记

特性

  • 自动垃圾回收
  • 更丰富的内置类型
  • 函数多返回值
  • 错误处理
  • 匿名函数和闭包
  • 类型和接口
  • 并发编程
  • 反射
  • 语言交互性

用途

  • 搭载 Web 服务器
  • 存储集群应用
  • 海量并行的支持

Hello World

运行 go 代码：go run hello.go

将源代码编译为二进制文件：go build hello.go

Go 语言结构

由包声明，函数，变量，语句&表达式和注释组成

Go 语言语法

  1. 标识符命名必须是由字母数字下划线组成，并且首位不能为数字

  2. 字符串拼接通过+实现

  3. 25 个关键字和 36 个预定义标识符

  4. 使用 fmt.Sprintf 格式化字符串并赋值给新串

Go 语言数据类型

布尔型，数字型，字符串型，派生类型（Pinter, Array, Struct, Channel, Func, Slice, Map, Interface）

数字型分为：整数型和浮点型，值得一提的是浮点型不仅支持 float32 和 float64 还支持 complex64 和 complex128，这将使得 go 语言成为科学计算选择的可能

Go 语言变量

  1. 变量声明分为 3 种情况

  • 指定变量类型，如果没有初始化，则变量默认为零值。

  • 根据值自行判定变量类型

  • 如果变量已经使用 var 声明过了，再使用 := 声明变量，就产生编译错误

特殊常量 iota

Go 语言运算符

  • & 返回变量存储地址 &a; 将给出变量的实际地址。

    • 指针变量。 \*a; 是一个指针变量

  • go 没有三目运算符

Go 语言条件语句

值得注意的是 select 语句，类似于 switch 语句，但是 select 会随机执行一个可运行的 case。如果没有 case 可运行，它将阻塞，直到有 case 可运行。

但是只能用于 channel 的操作(写入/读出)

关于 select： https://zhuanlan.zhihu.com/p/256950290

Go 语言循环控制

break, continue, goto
